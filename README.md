# JSCompiler

Javascript compiler in javascript for FEFU course.

## Install

Install ```NodeJS``` and ```npm```, then run in project directory

```
npm install -g yarn
yarn install
```

## Usage

Parse source files:
```
yarn start filePath1 filePath2 ...
```

Run tests:
```
yarn test
```
